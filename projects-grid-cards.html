DealUp


Website which can be used by shopkeepers, dealers and small manufacturers to keep records of payments and orders made by them. The user can build their profile on the website and manage their income and expenditure.
The website is built using HTML, CSS, Javascript and Bootstrap Studio. The graphics for the website were designed on Canva and Wix.
This project was made with my team members during the 36 hour long Delhi Hacks Hackathon held from 5th to 6th September, 2020. The project won the MLH Award for Best Beginner Hack.
